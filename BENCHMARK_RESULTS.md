## 6. 优化成果总览

| 优化项 | 技术 | 效果 | 文件 |
|--------|------|------|------|
| **ROPE 多片段拼接** | PHP ROPE opcode | concat-4: 14x慢→6.1x快 | CodeGenerator + runtime.h |
| **implode 两遍扫描** | O(N²)→O(N) | explode+implode **2-3x提速** | builtin.h |
| **explode 精确容量** | 预数分隔符 | 零 realloc | builtin.h |
| **对象复用池** | LIFO 128槽 | new+unset **36-52%提速** | object.h |
| **return 兼容性** | 零值匹配返回类型 | GCC/Clang 不再报错 | CodeGenerator |
| **JSON 位图+批量写入** | PHP json_encoder.c | json_encode 接近持平 | os/json.h |
| **数组池预热** | PHP zend_alloc | arr-create 12x慢→4.4x快 | runtime.h + array.h |

### 剩余优化空间

- **SSO 小字符串**: 24B 内联缓冲区，属性写入/字符串密集场景 2-3x
- **Arena Allocator**: 函数级 bump 分配器，全局 1.5-3x
- **`$a[]=` 语法**: 消除 array_push 函数调用开销
- **CodeGen 自动释放**: 对象重赋值时注入 release，消除手动 unset
